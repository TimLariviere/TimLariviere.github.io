With Xamarin, it is now easier than ever to share most of the logic between your apps. Most today's apps use a lot of data, so deciding how to store them locally can prove a challenge. SQLite provides a great way to store and query those easily and does support Xamarin.

In the original post, we'll see how to setup SQLite in a portable project (PCL) shared by two Xamarin projects (Android, iOS) and a UWP project in Visual Studio. The post can be found at [https://mcnextpost.com/2016/05/30/xamarin-using-sqlite-net-async-in-pcl/](https://mcnextpost.com/2016/05/30/xamarin-using-sqlite-net-async-in-pcl/)
